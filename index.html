 一个脚本运维面板

稳定 安全 高效 易用 简约 漂亮

开 始 使 用

面板开发引擎

Naive UI
Vue 3
TypeScript

后台管理面板采用前沿技术编写，界面简洁美观，功能强大。

脚本文件编辑

支持对文件内容进行在线编辑，编辑器采用自为 VS Code 提供支持的 Monaco Editor ，并且经过了我们的高度定制，提供众多语言解释器支持。

定时任务调度

通过数据表格页面进行管理操作，增删改查一应俱全，支持即时调试、查看历史日志、标签页分类等功能。

运行日志查询

支持使用文件名称搜索过滤、使用日期选择器组件进行指定日期范围查询，支持日志的高亮显示以及实时调整日志的字体大小。

文件维护管理

支持常见文件管理操作可管理项目的绝大部分文件，支持脚本在线调试运行并且可通过弹窗配置运行参数。

集成终端工具

集成基于 Xterm.js 的网页终端工具，实时、流畅，支持标签页多开。

文件内容对比

集成自 Monaco Editor 的专业差异编辑器，对比文件可选，支持多种对比模式。
