Message 消息中心

获取列表​

GET/message/v1/list

请求​

名称类型必填描述
limitstring否限制返回条数，默认10
statusstring<'read' | 'unread'>否状态过滤，支持多选（逗号分隔）
sourcestring<'system' | 'user'>否消息来源过滤
categorystring<'info' | 'warn' | 'error' | 'success'>否消息类型过滤
searchstring否搜索内容过滤，仅涉及消息标题和消息内容

响应​

获取分页​

GET/message/v1/page

请求​

名称类型必填描述
statusstring<'read' | 'unread'>否状态过滤，支持多选（逗号分隔）
sourcestring<'system' | 'user'>否消息来源过滤
categorystring<'info' | 'warn' | 'error' | 'success'>否消息类型过滤
searchstring否搜索内容过滤，仅涉及消息标题和消息内容
pagestring否指定页数，默认1
sizestring否指定分页大小数量，默认20
orderstring<'0' | '1'>否指定排序方式 0 升序 | 1 降序，默认降序

响应​

获取详情​

GET/message/v1/detail

请求​

名称类型必填描述
idstring是唯一标识

响应​

统计数量​

GET/message/v1/count

请求​

名称类型必填描述
limitstring否限制返回条数，默认10
statusstring<'read' | 'unread'>否状态过滤，支持多选（逗号分隔）
sourcestring<'system' | 'user'>否消息来源

响应​

更新状态​

POST/message/v1/status

请求​

名称类型必填描述
idnumber | number[]否唯一标识，若不提供id则表示操作全部消息
statusstring<'read' | 'unread'>是新的状态

响应​

删除​

POST/message/v1/delete

请求​

名称类型必填描述
idnumber | number[]是唯一标识

响应​

  • 获取列表
  • 获取分页
