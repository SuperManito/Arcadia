#### <GET>/cron/order/bindGroup</GET>

## 请求

无参数

## 响应

<APITable>

| 名称 | 类型 | 描述 |
| --- | :-: | --- |
| `bind` | `string` | 分类名称 |
| `count` | `number` | 任务数量 |

</APITable>

```json title="示例"
[
  {
    "bind": "",
    "count": 1
  },
  {
    "bind": "xxx_xxx",
    "count": 2
  }
]
```
