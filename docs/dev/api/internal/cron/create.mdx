title: 创建任务

import APITable from '@site/src/components/APITable';
import { POST } from '@site/src/components/RequestTag';

<POST>/cron</POST>

## 请求

<APITable>

| 名称 | 类型 | 必填 | 描述 |
| :-: | :-: | :-: | --- |
| `name` | `string` | 是 | 任务名称 |
| `cron` | `string` | 是 | 定时表达式 |
| `shell` | `string` | 是 | 执行的命令 |
| `active` | `number<0 \| 1>` | 否 | 状态 `1启用 \| 0禁用`，默认启用 |
| `remark` | `string` | 否 | 任务备注 |
| `type` | `string` | 否 | 任务类型过滤，当前固定分为 user 和 system |
| `bind` | `string` | 否 | 后端关联脚本参数 |
| `config` | `string` | 否 | 后端配置参数 |

</APITable>

## 响应

<APITable>

| 名称 | 类型 | 描述 |
| :-: | :-: | --- |
| `id` | `number` | 唯一标识 |
| `name` | `string` | 任务名称 |
| `cron` | `string` | 定时表达式 |
| `shell` | `string` | 执行的命令 |
| `type` | `string` | 任务类型，当前固定分为 user 和 system |
| `active` | `number<0 \| 1>` | 状态 `1启用 \| 0禁用` |
| `create_time` | `string` | 任务创建时间 |

</APITable>

```json title="示例"
{
  "active": 1,
  "create_time": "2024-01-01 00:00:00",
  "cron": "30 6 * * *",
  "id": 1,
  "name": "测试任务",
  "shell": "echo test",
  "type": "user"
}
```
