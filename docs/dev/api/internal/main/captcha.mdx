用于前端登陆表单

## 获取验证码图片

#### <GET>/captcha</GET>  

### 请求

<APITable>

| 名称 | 类型 | 必填 | 描述 |
| --- | :-: | :-: | --- |
| `w` | `string` | 否 | 图片宽度，默认120px |
| `h` | `string` | 否 | 图片高度，默认50px |
| `background` | `string` | 否 | 背景CSS |

</APITable>

## 判断是否需要验证码

#### <GET>/captcha/flag</GET>

### 请求

无参数

### 响应

- ContentType `application/json`

<APITable>

| 名称 | 类型 | 描述 |
| --- | :-: | --- |
| `showCaptcha` | `boolean` | 是否展示验证码 |

</APITable>

```json title="示例"
{
  "showCaptcha": false
}
```
