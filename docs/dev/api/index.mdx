### 支持的请求方法

<GET>（SELECT）从服务器取出资源</GET>
<POST>（CREATE）在服务器新建一个资源</POST>
<PUT>（UPDATE）在服务器更新资源</PUT>
<DELETE>（DELETE）从服务器删除资源</DELETE>
