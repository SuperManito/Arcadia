内部接口

本部分向你介绍项目所使用的内部接口

接口基准地址​

  • http://localhost:5678/api

身份认证​

在 Header 中添加 Authorization 字段，值为 Bearer {Token}

关于如何获取 Token 详见登录认证接口 详见此处

通用返回内容格式​

数据返回格式统一使用 JSON

参数说明
code业务代码
message消息
type类型
result返回结果（不一定有）

业务代码说明​

状态码说明
0请求失败
1请求成功
403认证失败

参数传递​

除 GET 请求通过 URL 传递参数外，其它请求方式均通过 Body 传递参数
