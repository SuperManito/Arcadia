登录认证

POST/user/auth​

请求​

名称类型必填描述
usernamestring是用户名
passwordstring是密码
captchastring是验证码

响应​

  • ContentType application/json
名称类型描述
tokenstringToken
newPwdstring系统自动更改的新密码
示例
{
  "token": "ABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789",
  "newPwd": ""
}

Token 有效时长：24小时

注释

newPwd 仅在系统强制更改用户密码后才返回，根据策略目前仅判断用户密码是否为初始密码，如果是则返回一个由随机字符组成的新密码，否则返回空字符串，密码变更后后台管理面板会有通知提醒。
