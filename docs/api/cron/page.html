分页查询

GET/cron​

请求​

名称类型必填描述
pagestring否指定页数，默认1
sizestring否指定分页大小数量，默认20
typestring否任务类型过滤，当前固定分为 user 和 system
searchstring否搜索内容过滤，仅涉及任务名称和任务执行命令
activestring否任务状态过滤，1启用、0禁用
tagsstring否标签过滤
remarkstring否备注内容过滤
sortstring否指定排序字段
orderstring否指定排序方式 0:升序 1:降序

响应​

  • ContentType application/json
名称类型描述
dataarray[object]定时任务数据项
pagenumber当前页码
sizenumber当前分页大小数量
totalnumber任务总数（数据库）

定时任务数据项

名称类型描述
idnumber唯一标识（逐一递增）
namestring任务名称
typestring任务类型，当前固定分为 user 和 system
cronstring定时规则（5位或6位的Cron表达式）
shellstring任务执行的命令
activenumber任务状态，1启用、0禁用
configstring后端配置参数
tagsstring任务标签
last_runtimestring最后一次运行的时间
last_run_usenumber最后一次运行的时长（单位秒，精确到小数点后3位）
sortnumber在列表中的位置序号（正整数），默认按照先后顺序递增
create_timestring任务创建时间
remarkstring任务备注
bindstring后端关联脚本参数
running_statusboolean当前运行状态（任务是否正在运行）
logPathstring后端关联日志路径（目录）
scriptPathstring后端关联脚本路径（文件）
