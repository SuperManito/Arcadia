#### <GET>/cron</GET>

## 请求

```mdx-code-block
<APITable>
```

| 名称 | 类型 | 必填 | 描述 |
| --- | :-: | :-: | --- |
| `size` | `string` | 否 | 分页数量，默认20 |
| `page` | `string` | 否 | 指定页数，默认1 |
| `search` | `string` | 否 | 搜索过滤内容，仅涉及任务名称和执行命令 |
| `bind` | `string` | 否 | 后端关联脚本参数过滤 |
| `tags` | `string` | 否 | 标签过滤 |
| `type` | `string` | 否 | 类型过滤 |

```mdx-code-block
</APITable>
```

## 响应

- ContentType `application/json`

```mdx-code-block
<APITable>
```

| 名称 | 类型 | 描述 |
| --- | :-: | --- |
| `id` | `number` | 唯一标识 |
| `name` | `string` | 任务名称 |
| `type` | `string` | 任务类型，当前固定分为 user 和 system |
| `cron` | `string` | 定时表达式 |
| `shell` | `string` | 执行的命令 |
| `active` | `number` | 任务状态，1启用、0禁用 |
| `config` | `string` | 后端配置参数 |
| `tags` | `string` | 任务标签 |
| `last_runtime` | `string` | 最后一次运行的时间 |
| `last_run_use` | `number` | 最后一次运行的时长（秒） |
| `sort` | `number` | 在列表中的位置序号，默认99999 |
| `create_time` | `string` | 任务创建时间 |
| `remark` | `string` | 任务备注 |
| `bind` | `string` | 后端关联脚本参数 |
| `running_status` | `boolean` | 当前运行状态 |
| `total` | `number` | 全部任务数量 |
| `size` | `number` | 分页数量，默认20 |
| `page` | `number` | 指定页数，默认1 |

```mdx-code-block
</APITable>
```
