#### <GET>/file/info</GET>

## 请求

```mdx-code-block
<APITable>
```

| 名称 | 类型 | 必填 | 描述 |
| --- | :-: | :-: | --- |
| `path` | `string` | 是 | 绝对路径 |

```mdx-code-block
</APITable>
```

## 响应

- ContentType `application/json`

```mdx-code-block
<APITable>
```

| 名称 | 类型 | 描述 |
| --- | :-: | --- |
| `fileSize` | `string` | 大小 |
| `birthtime` | `string` | 创建时间 |
| `atime` | `string` | 最近一次被访问的时间 |
| `ctime` | `string` | 目标对象的属性最近一次被修改的时间 |
| `mtime` | `string` | 目标对象的内容最近一次被修改的时间 |
| `name` | `string` | 名称 |
| `path` | `string` | 存储位置路径 |
| `mode` | `string` | 权限 |

```mdx-code-block
</APITable>
```
