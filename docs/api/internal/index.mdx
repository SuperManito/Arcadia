本部分向你介绍项目所使用的内部接口

### 接口基准地址

- `http://localhost:5678/api`

### 身份认证

在 Header 中添加 `Authorization` 字段，值为 `Bearer ${token}`

关于如何获取 `Token` 详见登录认证接口 [点击前往](/docs/api/internal/user/auth)
