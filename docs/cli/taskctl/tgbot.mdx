## 启动/重启服务

```bash
taskctl jbot start
```

## 停止服务

```bash
taskctl jbot stop
```

## 查看运行日志

```bash
taskctl jbot logs
```

## 查看错误日志

```bash
pm2 logs jbot
```

## 更新升级

```bash
taskctl jbot update
```
使用本地最新源码重装

:::caution 注意
执行安装操作后底层代码目前仅支持无缝迁移 **jbot/diy** 目录下的用户文件，请注意提前备份你放置在 **jbot** 目录下除 **jbot/diy** 子目录以外的其它重要文件
:::
