## 关于定时任务

定时规则优先从脚本备注内容中获取，由独特的算法提供支持，如若没有或者未识别到那么将指定每天执行一次的随机定时规则，你可以在之后使用后台管理面板来自定义修改

执行的命令为普通执行 `task run xxx` ，没有附加任何命令选项，如有需要请参考文档教程自行修改，可以直接调试

定时任务不会重复添加，不过在特殊环境下新增的定时任务会直接覆盖原有的定时任务，例如在迁移后的环境或重新克隆了仓库
