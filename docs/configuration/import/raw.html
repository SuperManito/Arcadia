远程文件

通过下载文件 Wget 的方式扩展本地脚本，存放在 /arcadia/raw 目录下

大部分脚本都位于 GitHub 仓库上，如果你的设备不能有效与其连通可以尝试使用 CDN 进行加速例如 jsDelivr，不过更新可能会因为缓存而存在一定的延迟
