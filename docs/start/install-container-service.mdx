## Docker Compose _（可选）_

:::info 这是什么？
Docker 官方推出的容器编排工具，用于批量部署、管理容器等操作，基于 Docker Engine 引擎
:::

最新的 Docker Compose V2 版本已经集成在 Cli 中，你只需要通过包管理工具安装 **docker-compose-plugin** 软件包即可，例如 `apt-get install -y docker-compose-plugin`
